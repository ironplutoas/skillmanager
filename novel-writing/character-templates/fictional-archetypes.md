## 少年逆袭型（成长热血型）

### 模板基本信息

| 字段 | 内容 |
|------|------|
| 历史人物姓名 | 虚构原型（综合 Naruto、哈利波特、陈长生等） |
| 时代/背景 | 虚构世界 |
| 身份定位 | 成长型主角 |
| 人物类型标签 | 少年型、逆袭型、热血型、成长型 |

### 核心性格原型

| 字段 | 内容 |
|------|------|
| 性格关键词 | 少年意气、逆袭决心、热血、不屈、纯粹、成长 |
| ★价值观 | 没有什么比证明自己、守护想守护的人更重要。 |
| ★行为底线 | 绝不放弃，绝不背叛同伴，绝不屈服于出身限制。 |
| 性格形成典型模式 | 因出身边缘形成逆袭决心；因少年理想形成热血；因困境磨练形成不屈。 |

### 人物目标与矛盾的典型模式

| 字段 | 内容 |
|------|------|
| 常见内在欲望 | 从边缘崛起，证明自己，守护同伴 |
| 常见核心矛盾 | 出身低微 vs 理想宏大；热血冲动 vs 现实险恶 |
| 典型挣扎方式 | 以加倍努力弥补不足，以热血对抗冷漠，以成长突破限制 |

### 人物弧光典型路径

| 字段 | 内容 |
|------|------|
| 常见成长轨迹类型 | 正向成长（从少年到强者） |
| 弧光类型建议 | 正向成长适合热血叙事 |

### 适用建议

| 字段 | 内容 |
|------|------|
| 适用题材 | 玄幻、奇幻、武侠（成长型主角） |
| 适用角色定位 | 主角（成长奋斗史） |
| 与什么世界观冲突最有效 | 与"血统等级"的世界冲突；与"安稳守旧"的价值观冲突 |
| 组合建议 | 与"稳重老成型"人物搭配（互补热血与稳重） |

### 经典案例参考

| 字段 | 内容 |
|------|------|
| 现有文学作品类似人物 | 《火影忍者》鸣人、《哈利波特》哈利、《择天记》陈长生、《将夜》宁缺 |

| 典型挣扎方式 | 以智谋影响局势，以隐忍等待时机，以不留痕迹的方式守护 |
