## 亚历山大型（征服雄心型）

### 模板基本信息

| 字段 | 内容 |
|------|------|
| 历史人物姓名 | 亚历山大大帝 |
| 时代/背景 | 古希腊，马其顿帝国 |
| 身份定位 | 军事家、征服者、帝王 |
| 人物类型标签 | 雄心型、征服型、天才型 |

### 核心性格原型

| 字段 | 内容 |
|------|------|
| 性格关键词 | 雄心、征服、果断、天才、不满足、远见 |
| ★价值观 | 没有什么比征服世界、证明自己超越前人更重要。 |
| ★行为底线 | 绝不满足于已有成就，绝不畏惧未知，绝不让阻碍者存活。 |
| 性格形成典型模式 | 因少年天才形成自信雄心；因父亲铺垫形成征服基础；因不满足形成持续扩张。 |

### 人物目标与矛盾的典型模式

| 字段 | 内容 |
|------|------|
| 常见内在欲望 | 征服已知世界，超越父亲成就，留下不朽名声 |
| 常见核心矛盾 | 雄心无止境 vs 人力有极限；征服欲 vs 部下疲惫 |
| 典型挣扎方式 | 以速度突破边界，以战绩激励追随，以远见规划下一步 |

### 人物弧光典型路径

| 字段 | 内容 |
|------|------|
| 常见成长轨迹类型 | 正向成长（从王子到大帝）或 悲剧型（英年早逝） |
| 弧光类型建议 | 正向成长适合征服叙事 |

### 适用建议

| 字段 | 内容 |
|------|------|
| 适用题材 | 玄幻（征服型主角）、历史、奇幻 |
| 适用角色定位 | 主角（征服史）、反派Boss（征服者）、天才型人物 |
| 与什么世界观冲突最有效 | 与"守旧安稳"的世界冲突；与"知足常乐"的价值观冲突 |
| 组合建议 | 与"稳重型"人物搭配（互补雄心与稳重） |

### 经典案例参考

| 字段 | 内容 |
|------|------|
