## 诸葛亮型（智谋忠诚型）

### 模板基本信息

| 字段 | 内容 |
|------|------|
| 历史人物姓名 | 诸葛亮 |
| 时代/背景 | 三国时期，蜀汉 |
| 身份定位 | 政治家、军事家、谋士 |
| 人物类型标签 | 智谋型、忠诚型、鞠躬尽瘁型 |

### 核心性格原型

| 字段 | 内容 |
|------|------|
| 性格关键词 | 智慧、忠诚、谨慎、鞠躬尽瘁、隐忍 |
| ★价值观 | 没有什么比忠君爱国、完成托孤重任更重要。 |
| ★行为底线 | 绝不背叛主公，绝不滥用权力谋私，绝不在未筹谋周全时轻举妄动。 |
| 性格形成典型模式 | 因受托孤之重，形成"以命相报"的责任感；因局势艰难，形成谨慎筹谋的习惯。 |

### 人物目标与矛盾的典型模式

| 字段 | 内容 |
|------|------|
| 常见内在欲望 | 实现理想（兴复汉室），完成承诺，留下清名 |
| 常见核心矛盾 | 理想宏大 vs 现实艰难；鞠躬尽瘁 vs 人力有时而穷 |
| 典型挣扎方式 | 以加倍筹谋弥补资源不足，以身先士卒激励士气，明知不可为而为之 |

### 人物弧光典型路径

| 字段 | 内容 |
|------|------|
| 常见成长轨迹类型 | 稳定型（坚守信念至终）或 悲剧型（理想未竟而逝） |
| 弧光类型建议 | 稳定型适合正面形象，悲剧型适合悲壮叙事 |

### 适用建议

| 字段 | 内容 |
|------|------|
| 适用题材 | 权谋、历史、玄幻（智囊角色） |
| 适用角色定位 | 主角（若写智谋奋斗史）、智囊型配角、导师型 |
| 与什么世界观冲突最有效 | 与混乱无序、背叛横行的世界冲突；与"利益至上"的价值观冲突 |
| 组合建议 | 与"理想主义型主角"搭配效果好，互补智谋与行动力 |

### 经典案例参考

| 字段 | 内容 |
|------|------|
| 历史事迹简述 | 白帝城托孤后独撑蜀汉，六出祁山，鞠躬尽瘁死而后已 |
| 现有文学作品类似人物 | 《庆余年》陈萍萍（忠诚智谋）、《琅琊榜》蔺晨 |
